# EndlessRunner
This is a game made for my "Intro to Game Programming" college course. This game is an endless runner, meant to teach me the basics on using GitHub versioning, animation, and game polish within Unity. Since I am really behind in this class, I plan to make the game over the next day or two (GameJam style), leaving me a little less than a week to make my final project. This game will be fairly similar to Google's Dinosaur game. Unlike that game, this game will have animations and also have the occasional obstacle that punishes the player for jumping (perhaps a tunnel where everywhere above the player is a wall that's all spiky)

I am using the "Platformer Tileset - Pixelart Grasslands" asset package made by BigManJD for the art
