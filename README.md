# EndlessRunner
This is a game made for my "Intro to Game Programming" college course. This game is an endless runner, meant to teach me the basics on using GitHub versioning, animation, and game polish within Unity. Since I am really behind in this class, over the next day or two, the plan is to make the game fairly similar to Google's Dinosaur game, leaving me about a week to make my final project. Unlike that game, this game will have animations and also have the ocassional obstacle that punishes the player for jumping (perhaps a tunnel where the wall is all spiky? IDK yet)

I am using the "Platformer Tileset - Pixelart Grasslands" asset package made by BigManJD for the art
